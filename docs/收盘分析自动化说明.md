# A股收盘持仓分析自动化

## 📋 任务说明

**目标**: 每个工作日下午3点A股收盘后，自动整理当天持仓表现并提出第二天投资建议。

---

## 🕐 运行时间

- **触发时间**: 工作日 15:00-15:30 (A股收盘后)
- **执行频率**: 每个交易日
- **节假日**: 自动跳过周末和法定节假日

---

## 📊 分析内容

### 1. 大盘表现
- 上证指数涨跌
- 深证成指涨跌
- 创业板指涨跌
- 市场情绪和成交量

### 2. 持仓股票表现
- 每只持仓股的当日涨跌
- 盈亏金额和盈亏比例
- 与成本价对比

### 3. 盈亏汇总
- 当日总盈亏
- 持仓分布统计
- 风险敞口分析

### 4. 明日投资建议
- 大盘研判
- 个股/ETF操作建议
- 风险提示

---

## 🛠️ 技术实现

### 脚本位置
```
/home/ubuntu/.openclaw/workspace-maxbot/scripts/daily-portfolio-analysis.sh
```

### 数据接口
- 腾讯股票API: `https://qt.gtimg.cn/q=股票代码`
- 实时获取股价、涨跌幅、成交量

### 输出格式
- Markdown报告
- 自动推送到GitHub
- 保存路径: `portfolio/daily-analysis-YYYYMMDD.md`

## 🚀 设置定时任务

### 方法1: 使用Cron (推荐)

```bash
# 编辑crontab
crontab -e

# 添加以下行 (工作日15:00运行)
0 15 * * 1-5 /home/ubuntu/.openclaw/workspace-maxbot/scripts/daily-portfolio-analysis.sh >> /home/ubuntu/.openclaw/workspace-maxbot/logs/daily-analysis.log 2>&1
```

### 方法2: 使用OpenClaw Heartbeat

在 `HEARTBEAT.md` 中添加：
```markdown
## 每日收盘分析
- 时间: 15:00-15:30
- 条件: 工作日
- 动作: 运行 daily-portfolio-analysis.sh
```
