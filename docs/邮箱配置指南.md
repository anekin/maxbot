# 邮箱配置指南

## 📧 当前配置

- **邮箱地址**: maxbot@yeah.net
- **SMTP服务器**: smtp.yeah.net
- **SMTP端口**: 465 (SSL)

## ⚠️ 需要完成的设置

由于网易邮箱（yeah.net）需要开启SMTP授权，请按以下步骤操作：

### 步骤1: 登录邮箱开启SMTP

1. 登录 yeah.net 邮箱网页版
2. 进入 **设置** → **POP3/SMTP/IMAP**
3. 开启 **SMTP服务**
4. 获取 **授权码**（不是邮箱密码）

### 步骤2: 更新授权码

获取授权码后，更新服务器上的密码文件：

```bash
# 在服务器上执行
echo "你的授权码" > /home/ubuntu/.openclaw/secrets/email_password.txt
```

### 步骤3: 测试邮件功能

```bash
python3 /home/ubuntu/.openclaw/workspace-maxbot/scripts/send-email.py test "maxbot@yeah.net"
```

## ❓ 常见问题

### Q: 为什么提示"用户无权登陆"？
A: 需要先在邮箱设置中开启SMTP服务，并使用授权码而非邮箱密码。

### Q: 可以发送到其他邮箱吗？
A: 可以，脚本支持发送到任意邮箱地址。

### Q: 一天可以发送多少封邮件？
A: 取决于邮箱服务商的限制，一般免费邮箱每天限制50-100封。
